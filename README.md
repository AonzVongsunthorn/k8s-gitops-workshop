# k8s-gitops-workshop

## Prerequisite

- kubeseal
- kube-secret-encode

Install kubesel

```bash
brew install kubeseal
```

Install kube-secret-encode

> Note: Ensure you have `~/go/bin` in your $PATH.

```bash
curl https://raw.githubusercontent.com/WhatTheFar/kube-secret-encode/master/install.sh | bash -s -- -b ~/go/bin
```
